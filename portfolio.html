I Tema 02 blev vi introduceret til CSS og begyndte at arbejde med wireframes og layoutdiagrammer for at planlægge strukturen af vores websites. Vi benyttede grid og flexbox for at arrangere elementerne effektivt og anvendte "mobile first"-metoden sammen med media queries for at sikre, at vores design var responsivt og fungerede på tværs af forskellige enheder.
Arbejdet blev udført i Visual Studio Code, og vi uploadede vores websites til vores domæne ved hjælp af FileZilla. I løbet af processen brugte vi også Firefox Developer Tools og funktionen "Inspicer" for at teste og justere koden direkte i browseren.
Gennem arbejdet med wireframes fik vi en bedre forståelse af, hvordan layout og design hænger sammen i praksis.
Temaet gav os både teknisk viden om CSS og webdesign samt en dybere forståelse af, hvordan man strukturerer et projekt på en velorganiseret måde, herunder korrekt opbygning af mapper og brug af webvenlige filformater.
